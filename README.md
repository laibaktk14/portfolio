# ✨ Laiba Javed | Magical Portfolio

> A magical, interactive portfolio website showcasing my skills, projects, and experience as a Computer Science student and Front-End Developer.

## 📖 About

This is my personal portfolio website built from scratch with passion and dedication. It showcases my journey as a Computer Science student at COMSATS University Islamabad and my work as a Front-End Developer intern at Optimus Automate.

The portfolio features a magical dark theme with interactive elements, smooth animations, and embedded project demonstrations.

## ✨ Features

### 🎨 Visual Features
- ✨ **Magical Dark Theme** with purple/blue gradients
- 🖱️ **Custom Cursor** with interactive hover effects
- 🌟 **Particle Background** with floating animations
- 💎 **Glassmorphism Cards** with backdrop blur
- 📱 **Fully Responsive** design for all devices

- 🧮 **Calculator Modal** - Fully functional calculator
- 🎵 **Music Player** - Interactive audio player
- 📧 **Contact Form** - With validation and toast notifications
- 📊 **Skill Progress Bars** - Animated on scroll
- 🔄 **Smooth Scrolling** - Navigation with animations

### ♿ Accessibility
- ARIA labels and roles
- Keyboard navigation support
